# Структура бэкенда

## Логические модули

1. **Модуль аутентификации и авторизации**
   - Регистрация пользователей (репетиторов и учеников).
   - Вход с выдачей JWT.
   - Проверка токена и ролей.

2. **Модуль управления учениками (CRM)**
   - CRUD для профилей учеников.
   - Поиск и фильтрация.
   - Привязка учеников к репетитору.

3. **Модуль материалов (файловое хранилище)**
   - Загрузка файлов (PDF, Word, изображения, видео).
   - Организация в папки.
   - Предоставление ссылок на файлы.
   - Управление доступом.

4. **Модуль тестов и заданий**
   - Создание тестов с вопросами разных типов.
   - Выдача заданий ученикам с дедлайном.
   - Приём ответов, автоматическая проверка, комментарии репетитора.

5. **Модуль расписания**
   - Создание событий (занятий) с привязкой к ученику.
   - Получение календаря.

6. **Модуль статистики**
   - Агрегация данных по результатам тестов.
   - Формирование данных для графиков.
