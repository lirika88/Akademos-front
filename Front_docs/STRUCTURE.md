## Основные части проекта (Структура приложения)

Проект представляет собой многофункциональный "рабочий стол" и делится на следующие логические блоки:

1.  **Модуль аутентификации:**
    *   Страницы входа и регистрации.
    *   Страница создания первого "профиля" репетитора.
2.  **Основное рабочее пространство:**
    *   Боковое меню навигации.
    *   Верхняя панель с быстрыми действиями и уведомлениями.
3.  **Модуль "Ученики" (CRM-часть):**
    *   Список учеников с поиском.
    *   Карточка ученика (личные данные, история занятий, текущие задания).
4.  **Модуль "Материалы" (Файловое хранилище):**
    *   Компонент файлового менеджера (как Google Drive): папки, файлы.
    *   Предпросмотр файлов.
5.  **Модуль "Тесты и задания":**
    *   Конструктор тестов: Динамические формы для создания вопросов.
    *   Редактор заданий: WYSIWYG-редактор (визуальный редактор текста) для описания домашки.
    *   Библиотека заданий: Список созданных шаблонов.
6.  **Модуль "Занятия" (Расписание):**
    *   Интеграция с календарем (FullCalendar или аналог).
    *   Карточка занятия (тема, ссылка на конференцию, прикрепленные тесты).
7.  **Модуль "Статистика":**
    *   Графики успеваемости учеников (на основе результатов тестов).
