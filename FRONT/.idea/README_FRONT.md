# Akademos - Фронтенд

Клиентская часть веб-приложения для организации работы репетитора.

## О проекте
Фронтенд представляет собой набор HTML-страниц, стилей и скриптов, формирующих личный кабинет репетитора. Проект разрабатывается как статическое веб-приложение без использования фреймворков.

## Функциональные разделы
- **Ученики** — просмотр списка, карточки учеников, статистика
- **Материалы** — файловое хранилище с поддержкой папок
- **Тесты** — конструктор тестов и заданий
- **Расписание** — календарь занятий
- **Статистика** — графики успеваемости

## Технологический стек
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript ES6+
- Fetch API для работы с сервером
- Git / GitHub



## Разработка
Проект использует Git Flow:
- `main` — стабильная версия
- `develop` — разработка
- `feature/*` — отдельные задачи

## Деплой
Автоматический деплой на Netlify при пуше в main.
